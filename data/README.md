#How to get the Kaggle Dataset

1. Visit the Kaggle competition page:
   https://www.kaggle.com/competitions/playground-series-s5e5/data

2. Click **Download All** to download the dataset.

3. Extract the downloaded files and place them in this 'data/' directory

You should have the following files:
- 'train.csv'
- 'test.csv'
- 'sample_submission.csv'
